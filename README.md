🎬 鑑賞ノート

映画・ドラマ・舞台・アートなどを観たあとに、
印象や気づきを記録するための Web アプリです。
三宅香帆さんの著書
『話が面白い人」は何をどう読んでいるのか』
で紹介されている「5つの技」メモ術を、
実験的にアプリ化しました。

🪞 概要

「比較・抽象・発見・流行・不易」の 5 つの視点をもとに、
作品を多角的に振り返ることができます。
日付・タイトル・感想・考察を入力して保存すると、
ローカル環境（ブラウザ内）に自動で記録されます。

✨ 主な機能
機能	説明
🧭 日付＋タイトル固定表示	長文を入力しても、作品タイトルと日付が常に画面上部に表示されます。
📝 感想入力欄	「面白かったシーン」「印象に残った部分」などを自由に記述。
🧩 5つの技ブロック	各ブロックで「技」「解釈」「感想」を記録（5セットまで）。
❓ 質問欄	自分への問いかけを残せます。
💾 自動保存	localStorage にデータを保持。再読み込みしても残ります。
🔍 検索・絞り込み	タイトル・感想・質問などのキーワード検索に対応。
📄 テキスト出力	各ノートを .txt ファイルとしてダウンロード可能。
🗑 削除	不要なノートを個別に削除できます。
📱 レスポンシブ対応	スマホ・タブレット・PC すべてで快適に操作可能。
🧱 技術構成

HTML / CSS / JavaScript (Vanilla)

localStorage API を使用（外部サーバー不要）

Google Apps Script や PHP 不要
→ ローカル環境 or 任意の Web サーバで動作します。

🪶 ファイル構成
project-root/

├── index.html

├── style.css

└── app.js


🚀 使い方

このリポジトリをクローンまたは ZIP でダウンロード

git clone https://github.com/yourname/kansho-note.git
cd kansho-note


index.html をブラウザで開く
（ローカルファイルとしてそのまま動作します）

日付・タイトルを入力し、感想や考察を記入

必須項目が埋まると「保存」ボタンが有効化されます

保存後は「鑑賞ノート一覧」に自動で追加

必要に応じて検索・テキスト出力・削除を利用できます

🧭 コンセプト

“観た作品の余韻を、そのまま記録に残す”

「思考の温度」を損なわずに記録できるよう、
淡い暖色のUIと、余白を意識したレイアウトにしています。
アプリ全体がノートのように“静かに書ける空間”であることを目指しました。
